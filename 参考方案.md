### 方案设计：视觉与文本融合的 RAG 架构

要让大模型在看图时能查阅 SOP，我们需要把传统 RAG 的“文本搜文本”升级为“图像搜文本”。整个过程分为三个极其连贯的步骤（总耗时可以控制在 2-3 秒内）：

#### 第一步：极速“看图识物”（Zero-shot 图像打标）

当用户上传图片后，不要一上来就做复杂的推理，而是先用极简的 Prompt 让大模型（**Qwen3-VL-4B**或者**Qwen2.5-VL**）快速提取图片的**核心物理特征**。

- **输入**：用户图片。
- **Prompt**：`“提取这张图片中的核心物件名称和场景（如：地下车库、金属杆、玻璃、大堂）。只需输出 3-5 个关键词。”`
- **大模型输出示例**：`["地下车库", "倒车杆", "变形", "黄黑相间"]`

#### 第二步：向量库精准检索（SOP RAG Retrieval）

拿到这几个关键词后，立刻去我们事先构建好的《三级保养类作业指导书》向量数据库中进行查询。

- **检索逻辑**：利用 `["地下车库", "倒车杆"]` 这样的关键词，向量库会瞬间匹配到《QHKC-WI-MT-01 基础设施管理作业指导书》中的原文。

- **召回的 SOP 知识片段**：

  > *提取自 MT-01 第 3.3 及 5.4 条*：车场基础设施包含排水道盖板、**倒车杆**、倒车镜、岗亭等。管理要求：**倒车杆**无明显锈蚀、**破损**，安装牢固。

#### 第三步：终极状态演绎与选项生成（注入 Prompt）

现在，我们把“原始图片” + **“召回的 SOP 标准”** 一起喂给你的核心判断逻辑（也就是咱们修改过的那版 Prompt），大模型就能像老专家一样下结论了。

- **动态组装的 Prompt 结构**：

  Markdown

  ```
  你是一个资深的企业设施维护专家。请分析图片，并参考以下【官方设施管理标准】：
  {从第二步检索到的 SOP 文本，如：车场基础设施包含倒车杆...要求安装牢固、无破损。}
  
  请观察图片，对比上述官方标准，找出设备偏差/故障。
  输出 1-3 个最可能的报修文案，公式：[可能的位置] + [具体物件] + [存在什么问题] + [需要怎么做/什么服务]
  ```

- **大模型最终输出（精准对齐企业话术）**：

  JSON

  ```
  {
    "internal_reasoning": "图片主体是黄黑相间的金属杆，背景是车库。根据 SOP 知识库，这是车场基础设施中的'倒车杆'。当前状态呈现弯曲变形，不符合 SOP 中'无明显破损、安装牢固'的要求。",
    "suggested_options": [
      {
        "category": "物理破损与脱落",
        "frontend_display_text": "车库倒车杆被撞弯变形，需要派人维修或更换。"
      },
      {
        "category": "物理破损与脱落",
        "frontend_display_text": "车库倒车杆底座松动不牢固，需要重新加固安装。"
      }
    ]
  }
  ```

### 如何处理这些 PDF 文件构建向量库？（代码实施指导）

为了让上述第二步能精准搜到内容，你需要把这些 PDF 变成高质量的索引。切记**不能按字数盲切**，而是要按“对象和标准”来提取。

你需要写一个 Python 脚本（可以借助 LangChain 和 PyPDFLoader）：

1. **文档清洗与提取**：针对《QHKC-WI-MT-01 基础设施管理作业指导书》，通过正则表达或大模型预处理，把它变成一条条“标准键值对”。
   - *原文*：“5.3.1 消防通道及公共通道：墙(地)面及天花无明显污迹及破损，管线整齐，照明灯具完好无损...”
   - *处理后入库的 Text*：`对象：消防通道/公共通道 (墙面, 地面, 天花板, 管线, 照明灯具)。正常标准：无污迹、无破损、管线整齐、灯具完好无损。故障特征：污迹, 破损, 管线散乱, 灯具破损。`
2. **向量化（Embedding）与入库**：将上面处理好的 Text 转化为向量（推荐使用 `bge-large-zh` 模型，对中文专业词汇理解极好），存入类似 Milvus、Qdrant 或 FAISS 这样的向量数据库中。

### 总结这个方案的优势

通过“先打标提词 -> 搜 SOP 标准 -> 融合图片出结论”这三步，大模型不再是凭空猜测。它是在**拿着你们前海嘉里中心的官方巡检标准在看图**。这不仅能极大地提高物件名称识别的准确率（比如不会把“倒车杆”叫成“挡车器”），还能让“需要派人做什么”这个后半句显得极其专业。
